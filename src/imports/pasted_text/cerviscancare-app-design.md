Buat desain UI/UX aplikasi mobile modern bernama CERVISCAN™ Care yang terintegrasi dengan produk CERVISCAN™ Dip Test untuk deteksi dini HPV/kanker serviks.
Gunakan gaya desain premium, futuristik, interaktif, minimalis, dan sangat user friendly dengan dominasi warna pink pastel, soft magenta, putih, serta aksen gradient pink-purple.
Tampilan harus clean seperti aplikasi kesehatan modern (FemTech + HealthTech), dengan elemen rounded, glassmorphism ringan, shadow halus, icon outline modern, dan animasi microinteraction.
